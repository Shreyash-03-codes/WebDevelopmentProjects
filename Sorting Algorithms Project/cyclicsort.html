Cyclic Sort:

        Definition:Cyclic Sort is a specialized sorting algorithm 
        that works efficiently for arrays where the numbers are in a specific range, 
        typically from 1 to n. It sorts the array by repeatedly placing each number 
        at its correct index (i.e., the value of each element corresponds to its 
        index in the sorted array). The algorithm works by cycling through misplaced 
        numbers until every element is in its correct position.

Intuition of Cyclic Sort:

Intuition:
        Cyclic sort operates under the assumption that the array contains integers 
        from 1 to n without duplicates. Instead of comparing and swapping adjacent 
        elements, the algorithm places each element in its correct position by 
        swapping it with the element currently occupying its target index.

Here is the Java code for Cyclic Sort along with an example: 

            import java.util.Arrays;

            public class CyclicSortExample {

                // Method to perform cyclic sort
                public static void cyclicSort(int[] array) {
                    int i = 0;
                    while (i < array.length) {
                        // Calculate the correct index for the current element
                        int correctIndex = array[i] - 1;
                        
                        // Swap the element if it's not in the correct position
                        if (array[i] != array[correctIndex]) {
                            int temp = array[i];
                            array[i] = array[correctIndex];
                            array[correctIndex] = temp;
                        } else {
                            i++;
                        }
                    }
                }

                public static void main(String[] args) {
                    // Example input array
                    int[] array = {3, 5, 2, 1, 4};
                    
                    System.out.println("Original array: " + Arrays.toString(array));
                    
                    // Call cyclic sort method
                    cyclicSort(array);
                    
                    // Output sorted array
                    System.out.println("Sorted array: " + Arrays.toString(array));
                }
            }

           Iteration 3:
           The element at index 0 is 5, which should be at index 4.
           Swap 5 with the element at index 4 (4).
          
           [4, 2, 3, 1, 5]

           Iteration 4:
           The element at index 0 is now 4, which should be at index 3.
           Swap 4 with the element at index 3 (1).
          
           [1, 2, 3, 4, 5]

           Final Array:
           Now, the array is sorted: [1, 2, 3, 4, 5].

Time Complexity:

            Best Case:O(n)
            Average Case:O(n)
            Worst Case: O(n)
            The algorithm achieves linear time complexity because, in each iteration,
            every number is placed in its correct position,making at most n-1 swaps for an array of length n.

Space Complexity:

            Space Complexity: O(1) (In-place sorting algorithm)
            Cyclic sort requires a constant amount of extra memory space, as it modifies the array directly.

Advantages:

  1. Linear Time Complexity: O(n) for sorting numbers within a known range.
  2. In-Place Sorting: Requires no extra space, making it space-efficient.
  3. Efficient for Known Range: Especially useful when sorting numbers between 1 to n in constant space.

Disadvantages:

  1. Limited Use Case:Works only when the array contains distinct elements from a known range (e.g., 1 to n).
  2. Fails with Duplicates or Non-Consecutive Numbers: Cannot handle arrays with duplicates or numbers outside the range.

Applications:

  1. Sorting Arrays of Range 1 to n: Best suited for sorting arrays where the values are consecutive numbers within a known range.
  2. Finding Missing or Duplicated Numbers: Cyclic Sort is often used in problems that involve finding missing numbers or duplicates in arrays with known ranges.
Note:Cyclic sort directly places elements into their correct positions with minimal swapping and achieves a sorted array in linear time.
