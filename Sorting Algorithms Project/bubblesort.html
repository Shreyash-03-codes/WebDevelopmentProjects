Bubble Sort:

        Definition:Bubble Sort is a simple sorting algorithm that
        repeatedly steps through thelist,
        compares adjacent elements, and swaps them if they are in the wrong 
        order. This process is repeated until the list is sorted. The algorithm 
        gets its name because smaller elements "bubble" to the top (beginning) of
        the list, while larger elements "sink" to the bottom.

Intuition of Bubble Sort:

Intuition:
        Bubble sort works by iteratively stepping through the list of data, comparing two
        adjacent items at a time. If they are in the wrong order, they are swapped. This 
        process repeats, and the largest unsorted element gradually moves to its correct 
        position, similar to bubbles floating to the surface of the water.

Here is the Java code for Bubble Sort along with an example: 
            import java.util.Arrays;

            public class BubbleSortExample {
                // Method to perform bubble sort
                public static void bubbleSort(int[] array) {
                    int n = array.length;
                    boolean swapped;
                    
                    // Loop through each element in the array
                    for (int i = 0; i < n - 1; i++) {
                        swapped = false;  // Flag to detect if a swap occurred in this iteration
                        
                        // Compare adjacent elements and swap if necessary
                        for (int j = 0; j < n - i - 1; j++) {
                            if (array[j] > array[j + 1]) {
                                // Swap the elements
                                int temp = array[j];
                                array[j] = array[j + 1];
                                array[j + 1] = temp;
                                
                                swapped = true;  // Set flag to true if a swap happened
                            }
                        }
                        
                        // If no swaps occurred, the array is already sorted
                        if (!swapped) break;
                    }
                }

                public static void main(String[] args) {
                    // Example input array
                    int[] array = {5, 1, 4, 2, 8};
                    
                    System.out.println("Original array: " + Arrays.toString(array));
                    
                    // Call bubble sort method
                    bubbleSort(array);
                    
                    // Output sorted array
                    System.out.println("Sorted array: " + Arrays.toString(array));
                }
            }

Time Complexity:

            Best Case (when array is already sorted): O(n)
            Average Case:O(n²)
            Worst Case: O(n²)
            Since bubble sort compares every adjacent element and can make up to n-1 comparisons per pass,
            it results in a quadratic time complexity for non-optimal cases.

Space Complexity:

                Space Complexity: O(1) (In-place sorting algorithm)
                Bubble sort requires a constant amount of extra memory space, as it modifies the array directly.

Advantages:

  1. Simplicity: Easy to understand and implement.
  2. In-place Sorting: No need for additional memory, other than the input array itself.
  3. Best Case Optimization: If the array is already sorted, bubble sort can finish early with O(n) time using an optimized version that detects when no swaps are needed.

Disadvantages:

  1. Inefficient: It has a quadratic time complexity (O(n²)) for average and worst cases, making it impractical for large datasets.
  2. Excessive Comparisons: Even when the list is mostly sorted, bubble sort will still compare adjacent elements unnecessarily unless optimized.

Applications:

  1. Educational purposes: Often used to teach basic sorting concepts due to its simplicity.
  2. Small datasets: Can be applied when working with very small lists where performance is not a primary concern.
  3. Partially sorted data: Optimized versions of bubble sort can be efficient when the dataset is almost sorted.
Note:While bubble sort is mostly a teaching tool and rarely used in practice for large datasets, its conceptual simplicity helps learners grasp sorting fundamentals easily.
